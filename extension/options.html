How to Use

1
Install the Native Host

Click Install Native Host above, download the binary, and run the install command in a terminal. Required to start the proxy server.

2
Install the CA Certificate (MITM mode only)

Download the CA cert above and install it in your proxy client app (not Chrome). Most apps have a custom CA / trusted certificates setting. This lets the proxy decrypt HTTPS to strip proxy headers.
